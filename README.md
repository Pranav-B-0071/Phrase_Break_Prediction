# Phrase_Break_Prediction
A Tamil phrase break prediction system for Text-to-Speech (TTS), developed as part of a MEITY-funded project. Uses POS tagging (Stanza), morphological features, and a CRF sequence model to assign prosodic break labels (NSIL, SSIL, MSIL, LSIL) to each word in a sentence enabling natural pause placement in synthesised Tamil speech.
